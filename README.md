# vuex-todo
A Todo app to learn vuex with vue for state management. The app uses JSONPlaceholder api.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
